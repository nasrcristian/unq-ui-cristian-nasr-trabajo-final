Para poder correr el proyecto:

1. Clonar el repositorio: 
```shell

2. Moverse a la carpeta src dentro del repositorio:

```

3. Instalar las dependencias:
```shell
npm install
```
   
4. Levantar el proyecto:
```shell
npm run dev
```
